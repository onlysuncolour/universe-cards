# Bug 报告模板

请尽量补全以下信息（越完整越容易快速修复）：

```text
1. 运行的是哪个场景？（路径）
2. 是按 F5 运行项目，还是 F6 运行当前场景？
3. 操作步骤是什么？（可编号）
4. 预期结果是什么？
5. 实际结果是什么？
6. Godot Output 完整报错是什么？（复制全文）
7. 是否开启 verbose log？
8. 当前 run_seed / battle_seed 是什么？
9. 当前启用了哪些 feature flags？
10. 是否能稳定复现？（每次/偶现）
11. 截图或录屏（如有）
12. replay 文件（如有，附路径或压缩包）
```
