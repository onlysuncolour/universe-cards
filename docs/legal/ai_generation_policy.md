# AI 生成素材策略（AI Generation Policy）

## 目标

- 记录 AI 生成素材的来源、模型信息与 prompt
- 降低侵权/来源不明风险
- 保证素材可复现（style_version / prompt_version / seed）

## 禁止

- 使用在世艺术家名字作为风格
- 直接复刻商业 IP 设计 / Logo / 水印
- 移除水印或规避授权限制

## 必须记录

- tool / model / model_version（如可得）
- prompt / negative prompt
- seed（如有）
- 输出文件路径（source 与 imported）
